吉林省卫生健康委关于新型冠状病毒肺炎疫情情况通报（2021年1月16日公布）

  • 来源：
  • 发布时间：2021-01-16
  • 字体：大 中 小

　　1月15日0-24时，全省无新增境外输入确诊病例。截至1月15日24时，全省连续259天无新增境外输入确诊病例。全省累计报告境外输入确诊病例19例，累计治愈出院19例。

　　1月15日0-24时，全省无新增本地确诊病例。截至1月15日24时，全省连续173天无新增本地确诊病例。全省累计报告本地确诊病例138例，累计治愈出院136例，病亡2例。

　　1月15日0-24时，全省新增无症状感染者13例，均与1月13日通报的黑龙江省输入无症状感染者3有关联，以上人员均在集中隔离医学观察期间通过主动筛查发现。

　　无症状感染者1：女，1938年出生，公主岭市人，住址为公主岭市范家屯镇老公安局家属楼。1月8日，在公主岭市范家屯镇艾尚瀚邦养生会馆参加1月13日通报的无症状感染者3的培训授课。1月14日在集中隔离医学观察期间核酸检测阳性，1月15日经临床专家组会诊，诊断为无症状感染者。

　　无症状感染者2：男，1957年出生，公主岭市人，住址为公主岭市范家屯镇范家屯实验中学西侧地房。1月8日，在公主岭市范家屯镇艾尚瀚邦养生会馆参加1月13日通报的无症状感染者3的培训授课。1月14日在集中隔离医学观察期间核酸检测阳性，1月15日经临床专家组会诊，诊断为无症状感染者。

　　目前，对以上13名无症状感染者已转运至当地定点医疗机构隔离医学观察。公主岭市正在对以上人员的密切接触者、次密切接触者和一般接触者开展追踪排查，并落实管控措施，对其生活和工作场所进行终末消毒。

　　提醒广大群众，如您是从黑龙江省有疫情发生地区及国内中高风险地区返（来）吉人员，或是与以上无症状感染者有接触的人员，必须主动向当地社区（村屯）报告，同时配合进行管控与核酸检测。为减少疫情传播风险，大家要不聚集、讲卫生、戴口罩、保持安全社交距离，一旦出现发热、咳嗽等急性呼吸道症状，请佩戴医用口罩及时到当地定点医疗机构发热门诊就诊。
